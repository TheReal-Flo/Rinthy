# Changelog

## 2.1.0

- iOS app builds are now available through Codemagic as an unsigned IPA for Sideloadly installation.
- Added iOS app metadata for OAuth callbacks and app icon generation.
- Notifications can now open related projects directly, including read-only access for projects you can view but do not manage.
- Projects without edit access now show only Overview and Versions, with create/edit/delete version actions hidden.
- Fixed organization projects so projects you do have access to still show the full management UI.
- Updated the glass theme. It is still not fully finished yet, but the current pass improves the direction.
- Updated the trend/analytics experience.
